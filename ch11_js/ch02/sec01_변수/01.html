자바스크립트 변수 정리

1. 변수란?

- 데이터를 저장하기 위해 이름을 붙여둔 공간
- 값이 변할 수 있음

2. 변수 선언 규칙

  • 영문자, 숫자, _, $ 사용 가능
  • 숫자로 시작할 수 없음 (예: 1name ❌)
  • 공백 사용 불가 (예: user name ❌)
  • 대소문자 구분 (user ≠ User)
  • 예약어(예: if, for, let) 사용 불가

3. 변수 선언 방법

1) var    → 함수 범위(function scope), 재선언 가능 (권장 ❌)
2) let    → 블록 범위(block scope), 재선언 불가 (권장 ✅)
3) const  → 블록 범위(block scope), 상수(값 변경 불가) (권장 ✅)
